Getting Started with GitHub

Ready to stop saving files as `file1.bak` or want to collaborate more easily with your team?
It’s time to get on GitHub, and it’s really easy to get started!

  • Sign up for a free GitHub account.

    This is the easiest step in the process, we promise.

  • Maybe create a repository

    If you're going to use GitHub to host code that you've written, you need to create a repository. Follow the instructions to push code to your new repository, and see where your code will live on GitHub.

  • Make your first contribution!

    If you're contributing to an existing project instead, head over to that repository, see if they have any specific instructions for contributors, and get to coding!
